MySql
1. Activate all necessary MySql programs.
2. Edit "Config.php" to desired specifications.
3. Access "index.php" from defined BASE_URL in "Config.php". To delete database, use "DeleteDB.php".

Simpletest
1. Go to "test.php".
2. Create a custom class that extends UnitTestCase.
3. Create a custom function that includes the file to be tested.
4. Assert what you want tested and the expected result.
5. Run "test.php" from browser.
